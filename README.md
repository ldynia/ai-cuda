Project for testing various LLMs on CUDA vs. CPU

## Requirements

```bash
sudo apt-get install nvtop
```

## Install


```bash
pyton3 -m venv .venv

source .venv/bin/activate
deactivate

pip install -r requirements.txt
```

## Run

```python
python3 semantic-analysis.py
python3 speach-recognition.py
```
